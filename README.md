# DCA Travel — Portal B2B para Agencias de Viaje

Una plataforma web diseñada para fortalecer la presencia digital de **DCA Travel** y ofrecer a las agencias de viaje una experiencia más clara, profesional y útil al momento de explorar itinerarios, destinos y materiales comerciales.

Este proyecto nace con una idea sencilla:
**hacer que vender viajes sea más fácil, más visual y más confiable para las agencias.**

---

## ✨ Visión del proyecto

DCA Travel no busca ser solo una página informativa.
La intención es construir un ecosistema digital B2B donde las agencias puedan encontrar inspiración, consultar itinerarios, acceder a materiales de apoyo y, en una siguiente etapa, gestionar recursos personalizados para sus propios clientes.

El objetivo principal es combinar:

* Diseño limpio y profesional.
* Experiencia de usuario clara.
* Material turístico accesible.
* Identidad visual confiable.
* Herramientas útiles para agencias de viaje.

---

## 🧭 Características principales

* Catálogo de itinerarios turísticos.
* Páginas de detalle por destino.
* Diseño responsive para desktop, laptop, tablet y móvil.
* Modo claro y modo oscuro.
* Secciones informativas sobre DCA Travel.
* Material visual pensado para agencias.
* Interfaz limpia con enfoque B2B.
* Componentes reutilizables.
* Base preparada para futuras integraciones con backend y CMS.

## 🚀 Instalación y ejecución local

Clona el repositorio:

```bash
git clone <url-del-repositorio>
```

Entra al proyecto:

```bash
cd dca-travel
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el entorno de desarrollo:

```bash
npm run dev
```

Abre el proyecto en el navegador:

```bash
http://localhost:3000
```
